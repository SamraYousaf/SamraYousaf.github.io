Recent Work

Desktop Application for Job Shop Scheduling Optimization

Designed and implemented a Python-based Genetic Algorithm (GA) to minimize the Makespan in standard job shop scheduling scenarios. The solution incorporates a local optimization technique, Neighborhood Search Heuristics, to improve convergence speed. Performance was validated against standard Muth & Thomson and Lawrence benchmarks. The tool features a user-friendly GUI that supports both predefined and custom benchmark inputs, and generates outputs including optimized Makespan, Gantt charts, and convergence plots. Impressively, the algorithm outperformed several existing solutions in terms of scheduling efficiency.

ADAS L2 Web Application for Module Simulation and Interface Testing

I customized web based GUI for testing and visualizing key ADAS modules, including Adaptive Cruise Control (ACC), Lane Departure Warning (LDW), Collision Avoidance, and Emergency Braking. The application was built using Flask and Socket.IO for real time communication, with a focus on frontend responsiveness, usability, and modular integration. The interface served as a simulation and debugging platform, enabling team members to visualize sensor fusion outputs and vehicle state transitions during L2 autonomous driving scenarios. Through this project, I contributed to creating a reliable testing environment that supports rapid iteration, module validation, and driver system interaction analysis essential for the development and verification of safe autonomous systems.

Situation(SIT) Module Design and Testing for Adaptive Cruise Control (ACC) in ADAS Systems

This project focused on the design, testing, and integration of the Situation (SIT) module within the Adaptive Cruise Control (ACC) pipeline of an L2 ADAS system. The SIT module was responsible for intelligent target selection, dynamic lane assignment, and real time decision making based on sensor fusion inputs. The implementation involved building object filtering logic, handling classification based decisions, and applying contextual rules to identify suitable ACC targets. Complex scenarios such as pedestrian filtering at high speeds and radar-camera confirmation were encoded through decision tables and control structures in C++. The project also included dynamic lane assignment using both egomotion and road model trajectories, followed by precise Frenet coordinate transformations for final target validation. SIT design, robust unit testing and debugging were conducted to ensure correct SIT behavior and reliable target passing to downstream ACC logic, contributing to safer longitudinal control in autonomous driving simulations.

Design and Manufacturing of a Passive Upper Limb Exoskeleton for Industrial Load Support

This project involved the design, modeling, and fabrication of a passive exoskeleton system aimed at reducing upper limb fatigue for workers in industrial environments. The exoskeleton was engineered to support the human arm while lifting or holding loads up to 15 kilograms, using a spring based mechanism as the primary actuation source. Without relying on electrical power or motors, the system provided mechanical assistance through isoelastic support, helping distribute gravitational forces and minimize muscular effort. The structural framework was designed in SOLIDWORKS and fabricated using lightweight materials, while MATLAB simulations were used to analyze the biomechanical impact of the spring mechanism on load reduction. The project demonstrated how passive wearable technology can improve ergonomics, reduce the risk of strain injuries, and enhance long duration task performance in manual labor settings.

Mechatronics Lab Inventory and Navigation GUI

This project involved the design and development of a desktop based graphical user interface (GUI) application to visually represent and navigate the various laboratories of the Mechatronics and Control Engineering Department. Developed using Python tkinter library, the application features a clean layout with department branding, background imagery, and interactive buttons for accessing detailed lab sections. The GUI acts as a digital front desk displaying an image of the department building, providing easy access to individual lab categories (such as robotics, hydraulics, instrumentation, and simulation), and enabling a modular expansion to new lab entries. To enhance user experience, the interface was designed with a clear visual hierarchy, responsive layout, and organized frames for image rendering and control logic. The application can be extended to support features like lab attendance logging, student access records, and resource management. This project showcases my early work in GUI programming, user-centered design, and integration of visual assets in Python applications.
